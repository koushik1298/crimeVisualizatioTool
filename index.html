USA Crime Data Visualization

ChoroplethMap

Choropleth map of crime rates in the United States. The code uses D3.js library to visualize the data and a slider to enable the user to interact with the map and select a year. The map is split into two sections, one for violent crime rates and one for non-violent crime rates.

Go to ChoroplethMap

Bubble Chart

Chart is to display the relationship between the total crime and population of each state in the USA over time. The user can interact with the chart by sliding a year range input, updating the chart to show the data for the selected year.
